# Node Auth - Clean Architecture

## Este proyecto está organizado siguiendo principios de Clean Architecture y está escrito en TypeScript.

### Requisitos

- Node.js 22.14+

### Instalación

#### Instala dependencias

```bash
pnpm install
```

#### Iniciar aplicacion

```bash
pnpm run dev
```

### Comandos útiles

#### Iniciar la aplicación en modo ejecución con `tsx` (asume que el entrypoint es `src/index.ts`)

```bash
pnpm start
```

#### Compilar el proyecto con TypeScript (usa `tsc` según `tsconfig.json`):

```bash
pnpm run build
```
